# Frenciniz — Fren Aksamı E-Ticaret Sitesi

Otobüs, kamyon, tır ve dorse fren aksamı ürünleri için e-ticaret platformu.

**Dumanlar Ticaret** çatısı altında faaliyet göstermektedir.

## Kurulum

```bash
npm install
npm run dev
```

## Özellikler

### Müşteri Tarafı
- Ürün listeleme, filtreleme ve sıralama
- Ürün detay sayfası (çoklu görsel galerisi + zoom)
- Sepet, kupon kodu, ücretsiz kargo barı
- 3 adımlı checkout (Garanti BBVA / PayTR / Param sanal pos)
- Üye kayıt / giriş (telefon numarası ile)
- Hesabım (siparişler, adresler, bildirimler, şifre değiştir)
- Favoriler ve son görüntülenenler
- Canlı destek chat widget
- WhatsApp butonu
- Stok alarmı (tükenince haber ver)
- Sık alınanlar (tekrar sipariş)
- Mobil uyumlu (hamburger menü + filtre drawer)
- Çoklu dil desteği (TR / EN)
- Çerez bildirimi (KVKK uyumlu)
- Bülten aboneliği

### Bilgi Sayfaları

- İade Politikası
